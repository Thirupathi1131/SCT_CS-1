# 🔐 Caesar Cipher Tool

## 📌 Description
This project implements the Caesar Cipher algorithm to encrypt and decrypt text using a shift value.

The Caesar Cipher is one of the simplest and most widely known encryption techniques in cryptography.

---

## 🚀 Features

- 🔒 Encrypt text using shift value  
- 🔓 Decrypt text using shift value  
- 👤 User input for message and key  
- ⚡ Real-time output display  
- 🎨 Simple and clean UI  

---

## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript  

---

## 💡 How It Works

- Each letter in the message is shifted by a fixed number (shift value)
- Encryption moves forward in alphabet  
- Decryption moves backward  

Example:  
HELLO → Shift 3 → KHOOR
